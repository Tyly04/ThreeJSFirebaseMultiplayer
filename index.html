ThreeJSFirebaseMultiplayer

A Three JS Multiplayer game using Google Firebase, by Me, Tyly04. Just so you know, I didn't add the walking around and collision detection. This was thanks to: Web Maestro.

I got the inspiration from jgordon510 and Firebase Asteroids. I used the 3D Javascript Engine ThreeJS by Mr. Doob. You can play the game (Bug testing, in progress) here.
